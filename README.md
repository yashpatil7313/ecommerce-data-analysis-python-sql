# E-commerce Data Analysis using Python and SQL

## Project Overview
This project analyzes an e-commerce dataset using SQL queries and Python for data visualization and analysis.

The goal of the project is to extract business insights from the dataset such as:
- Customer distribution
- Order trends
- Sales by product category
- Payment installment analysis

## Tools Used
- Python
- SQL
- Pandas
- Matplotlib
- Seaborn
- MySQL / PostgreSQL

## Analysis Performed
1. List unique customer cities
2. Count number of orders placed in 2017
3. Calculate total sales per category
4. Analyze payment installments
5. Customer distribution by state

## Learning Outcome
This project helped me practice:
- Writing SQL queries
- Connecting Python with SQL databases
- Performing data analysis using Pandas
- Creating visualizations
